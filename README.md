# 🧠 Quiz Application

A simple and interactive Quiz Application built using **HTML, CSS, Bootstrap, and JavaScript**.
This project allows users to answer multiple-choice questions and get their final score instantly.

## 🚀 Features

* 📌 Multiple-choice quiz questions
* ⏱️ Question counter display (e.g., 1/10)
* 🎯 Instant answer selection
* 📊 Final score display
* 🔄 Restart quiz option
* 💡 Clean and responsive UI using Bootstrap

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (Vanilla JS)
